# Классификация ирисов
## Домашнее задание 1

Классификация видов ирисов (Iris-setosa, Iris-versicolor, Iris-virginica) по морфологическим признакам.

## Модели
- Логистическая регрессия
- KNN (k=1, k=5)
- Decision Tree (max_depth=5, max_depth=10)
- Random Forest
- SVM

## Результаты
Лучшая модель: Decision Tree (max_depth=10) - Accuracy: 0.956

## Зависимости
- scikit-learn
- pandas
- matplotlib
- seaborn
- datasets (Hugging Face)
